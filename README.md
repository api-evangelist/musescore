# MuseScoreThis is a repo for managing machine readable artifacts for the [MuseScore APIs](https://musescore.org/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [MuseScore APIs](https://musescore.org/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://musescore.org/)- [Developer](https://musescore.org/)- [Twitter](https://twitter.com/musescore)- [GitHub](https://github.com/musescore)The [APIs.json](https://github.com/api-evangelist/musescore/blob/master/apis.json) index provides links to all the relevant links for the MuseScore APIs, including the OpenAPI and Postman if they are present.
